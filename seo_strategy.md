# SEO Strategy

## In scope
- Public marketing page (`/`)
- Public tool routes exposed without authentication (`/upload`, `/builder`, `/preview`)
- Static crawlability files for the public app (`robots.txt`, sitemap, favicon, llms.txt)

## Out of scope
- API implementation details that do not affect public crawlability or metadata
- Internal development artifacts unrelated to the deployed public app

- Users looking for ATS-friendly resume building help
